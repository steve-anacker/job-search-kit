# job-search-kit

Free tools and standards from an active job search — built with Claude, used daily.

---

## What's here

### `writing-craft-standards.md`

A guide for making AI-assisted professional writing sound human rather than generated.

Covers:
- Hard-ban vocabulary (the words that instantly signal AI)
- Structural anti-patterns (LLM sentence rhythm, punchy closers, stacked abstraction)
- Career narrative cliches
- Long-form and social content tells (LinkedIn posts, blog posts)
- A pre-output self-check you can run before any draft
- Anti-overcorrection warning — how to avoid the new recognizable style of "anti-AI writing"

The goal is not "sound unlike AI." The goal is "sound like someone who actually did the work."

**How to use it with Claude:**

Drop `writing-craft-standards.md` into your Claude project as a reference document. In any session:

```
Apply the writing standards file before generating anything.
```

Or add it to your system prompt:

```
Before writing any professional content, check against writing-craft-standards.md.
```

Pairs with a personal voice profile for user-specific tone and sentence preferences.
This file covers universal AI tells. A voice profile covers how *you* specifically sound.
